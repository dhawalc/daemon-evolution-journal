## What's Actually Happening

Every 15-18 minutes:

1. **PROBE** runs 7 capability tests with ground-truth validators
2. **JUDGE** scores each result (0.0-1.0)
3. **GAP DETECTOR** finds the highest-priority bottleneck
4. **INTERVENTION** runs a time-boxed improvement task (5 min max)
5. **AUDIT** checks for stagnation, regression, or busywork patterns

If Audit detects busywork (same score for 5+ cycles), it forces exploration.

## Code Snapshot

```python
# consciousness/loop.py — The Soul
class ConsciousnessLoop:
    def cycle(self):
        # 1. Measure ground truth
        probe_results = self.probe.run_all()
        
        # 2. Score with deterministic validators  
        scores = self.judge.score(probe_results)
        
        # 3. Find highest-priority gap
        gap = self.gap_detector.find_gap(scores)
        
        # 4. Execute time-boxed improvement
        result = self.intervention.execute(gap)
        
        # 5. Check for pathological patterns
        self.audit.check(scores, result)
```
